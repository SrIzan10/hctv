---
title: How to stream
description: Get started with OBS and streaming on hackclub.tv
---

- open obs
- open settings
- open "Stream"
- set service to custom
- set url to rtmp://hackclub.app:45913/live
- on the website, click "Regenerate key"
- paste the key into your obs "Stream Key"
- start streaming!!
